## Opis projekta

**Project Mnemosyne** je mali side project, cli program za čuvanje lozinki.

- Program će biti **100% siguran**.
- Projekt je pod **MIT licencom**, što znači da ga bilo tko može koristiti, distribuirati i modificirati.
- Neću često raditi na ovome zbog obaveza na fakultetu, ali ću se potruditi raditi na projektu tu i tamo.
